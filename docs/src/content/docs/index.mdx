## What is Lexicon?

Lexicon is a contract-driven verification tool for Rust. It lets you declare what your code must do, must not do, and what is out of scope -- then continuously verifies that your implementation matches.

Lexicon works with or without AI. When used with Claude Code, it syncs your contracts and verification context into `CLAUDE.md` so the AI always operates within defined boundaries.

## Core Workflow

## Key Features

- **Behavioral contracts** -- TOML-based specifications with invariants, required/forbidden semantics, and edge cases
- **Conformance suites** -- auto-generated test harnesses from contracts
- **Weighted scoring** -- multi-dimensional health scores with pass/warn/fail verdicts
- **Verification gates** -- shell commands (cargo fmt, clippy, tests) run as subprocesses
- **AI integration** -- optional, bounded, audited; syncs context to CLAUDE.md
- **Audit trail** -- every verification run and change is recorded as JSON
- **Terminal UI** -- tab-based dashboard for contracts, gates, and scores
