title: lexicon behavior

---

## Subcommands

### `lexicon behavior add`

Add a behavior scenario to a contract.

```bash
lexicon behavior add <contract_id>
```

Creates a BDD-style scenario in `specs/behavior/` with Given/When/Then format.

### `lexicon behavior sync`

Sync behavior scenarios when contracts change.

```bash
lexicon behavior sync
```

### `lexicon behavior generate`

AI-generate behavior scenarios from a natural language description.

```bash
lexicon behavior generate "<intent>"
```

Requires authentication (`lexicon auth login`).

**Example:**

```bash
lexicon behavior generate "user session expiration after inactivity"
```

Generates a markdown behavior scenario file with:

- Main success path (Given/When/Then)
- Failure and edge case scenarios
- Links to contract clauses where applicable

The generated file is shown as a patch preview. You review and accept or reject before it is written to `specs/behavior/`.
