## Core Commands

| Command | Description |
|---------|-------------|
| `lexicon init` | Initialize lexicon in a repository |
| `lexicon verify` | Run full verification pipeline |
| `lexicon doctor` | Check repository health |
| `lexicon tui` | Launch the terminal UI dashboard |
| `lexicon improve` | AI-guided improvement loop |

## Contracts

| Command | Description |
|---------|-------------|
| `lexicon contract new` | Create a new contract through interactive workflow |
| `lexicon contract edit` | Edit an existing contract |
| `lexicon contract list` | List all contracts in the repository |
| `lexicon contract lint` | Check contracts for structural issues |

## Conformance

| Command | Description |
|---------|-------------|
| `lexicon conformance add <id>` | Generate conformance harness from a contract |
| `lexicon conformance sync` | Regenerate harnesses after contract changes |

## Scoring

| Command | Description |
|---------|-------------|
| `lexicon score init` | Create the scoring model |
| `lexicon score explain` | Display score breakdown with explanations |

## Gates

| Command | Description |
|---------|-------------|
| `lexicon gate init` | Create the default gates configuration |

## Coverage

| Command | Description |
|---------|-------------|
| `lexicon coverage report` | Show contract coverage report |

## API

| Command | Description |
|---------|-------------|
| `lexicon api scan` | Extract public API surface |
| `lexicon api diff` | Compare current API against baseline |
| `lexicon api report` | Produce API stability report |

## Sync

| Command | Description |
|---------|-------------|
| `lexicon sync claude` | Generate/update CLAUDE.md with managed blocks |

## Authentication

| Command | Description |
|---------|-------------|
| `lexicon auth login` | Authenticate with an AI provider |
| `lexicon auth status` | Show current authentication status |
| `lexicon auth refresh` | Refresh authentication tokens |
| `lexicon auth logout` | Clear stored credentials |

## Architecture
